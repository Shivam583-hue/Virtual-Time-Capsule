# **Samay Capsule**

Samay Capsule is a virtual time capsule website, a time capsule can be used for various purposes, including preserving personal memories, documenting historical events, or creating educational projects.
A virtual time capsule unlike a physical capsule uses a database to store images, text and other media.

---

## **🌟 Key Features**
1. **Create a time capsule and add texts and photos in it, and set a timer for its unsealing.**    

2. **You will be notified when the time capsule is unsealed**  

3. **Share your unsealed time capsule on any social media by just coping a post that will be generated after unsealing **

## **💻 Tech Stack**
- **Language & Runtime:** TypeScript, Node.js  
- **Backend:** Express (REST APIs)  
- **Frontend:** React  
- **Database & ORM:** PostgreSQL (NeonDB), Drizzle  
- **CSS Framework:** Tailwind CSS  
- **UI Libraries:** ShadcnUI  
- **Monitoring and logging:** Prometheus and Grafana
- **Web Push Notifications:** Web Push npm library
- **Auth:** Google O-Auth Provider  
- **Deployment:** Render _(Free Tier)_
